# SUNBAY SoftPOS 用户指南

## 概述

本文档提供 SUNBAY SoftPOS 系统的使用说明，包括管理后台和 Android 应用的操作指南。

---

## 1. 管理后台使用

### 1.1 登录

访问管理后台 URL，使用管理员账号登录。

### 1.2 设备管理

#### 设备审批
1. 进入"设备管理" > "待审批设备"
2. 查看设备信息和安全评分
3. 点击"批准"或"拒绝"

#### 设备监控
1. 进入"设备管理" > "设备列表"
2. 查看设备状态和健康评分
3. 点击设备查看详细信息

### 1.3 威胁管理

1. 进入"威胁管理"
2. 查看威胁事件列表
3. 处理威胁事件

### 1.4 交易查询

1. 进入"交易查询"
2. 设置查询条件
3. 查看交易记录

---

## 2. Android 应用使用

### 2.1 设备注册

1. 首次启动应用
2. 输入商户信息
3. 等待审批

### 2.2 交易处理

1. 输入交易金额
2. 点击"开始交易"
3. 等待健康检查
4. 刷卡或插卡
5. 输入 PIN（如需要）
6. 等待交易结果

详细使用指南请参考：
- [Android Demo 指南](../../sunbay-softpos-android/TRANSACTION_TOKEN_DEMO_GUIDE.md)
- [Frontend Demo 指南](../../sunbay-softpos-frontend/DEMO_GUIDE.md)

---

## 3. 常见问题

### 3.1 设备注册失败

- 检查网络连接
- 检查设备是否被 Root
- 联系管理员

### 3.2 交易失败

- 检查设备状态
- 检查健康评分
- 查看错误信息

# SUNBAY SoftPOS 术语表

## 技术术语

### A

- **API**: Application Programming Interface，应用程序编程接口
- **Attestation**: 设备认证，验证设备完整性的过程

### B

- **BDK**: Base Derivation Key，基础派生密钥
- **Backend**: 后端服务

### D

- **DUKPT**: Derived Unique Key Per Transaction，每次交易派生唯一密钥
- **DTO**: Data Transfer Object，数据传输对象

### E

- **EMV**: Europay Mastercard Visa，支付卡标准

### H

- **HSM**: Hardware Security Module，硬件安全模块

### I

- **IPEK**: Initial PIN Encryption Key，初始 PIN 加密密钥

### J

- **JWT**: JSON Web Token，JSON 网络令牌

### K

- **KSN**: Key Serial Number，密钥序列号

### M

- **MPoC**: Mobile Point of Commerce，移动商业终端

### N

- **NFC**: Near Field Communication，近场通信

### P

- **PCI**: Payment Card Industry，支付卡行业
- **PIN**: Personal Identification Number，个人识别码
- **PINPad**: PIN 输入设备

### R

- **REST**: Representational State Transfer，表述性状态转移
- **RSA**: Rivest–Shamir–Adleman，一种公钥加密算法

### S

- **SDK**: Software Development Kit，软件开发工具包
- **SoftPOS**: Software Point of Sale，软件销售终端
- **SQLite**: 轻量级关系数据库

### T

- **TEE**: Trusted Execution Environment，可信执行环境
- **TLS**: Transport Layer Security，传输层安全

### W

- **WebSocket**: 全双工通信协议

---

## 业务术语

- **设备注册**: 设备首次接入系统的过程
- **设备审批**: 管理员审核设备注册请求
- **健康检查**: 定期检查设备安全状态
- **威胁检测**: 识别设备安全威胁
- **交易鉴证**: 交易前验证设备状态
- **密钥注入**: 向设备注入加密密钥
- **密钥派生**: 从主密钥派生工作密钥
